Lists all included files and their purpose:

Library of Copilot Prompts.docx - it documents all Copilot prompts used in each task

Task 1: Email summary, Supply Chain Issues Review.docx. - it includes the email summary and the Supply Chain Issues Review report

Task 2: Meeting Invite and Agenda (Supply Chain Issues Review) and Action Plan.docx. - it includes screenshot of the meeting invite plus meeting summary and action plan 

Task 3: Supply chain risk assessment data Supply Chain Issue Analysis Report.docx. - it includes the Supply Chain Analysis and Risk Mitigation Planning

Task 4: Stakeholder presentation slides with speaker notes and data visualizations Supply Chain Issue Compliance Report.pptx. - it is a presentation in ppt for the business review with key issues, insight analysis, proposed solutions and implementation plan

Provides an overview of each task and its deliverables:
Task 1: 
The email summary identifies three key issues and provides five actionable next steps.
The Supply Chain Issues Review document is formatted with a consistent layout and fonts.
The document includes the following required sections: Introduction, Outline of Business Problem, Action Items, and Conclusion.

Task 2:
Create a meeting invite titled “Supply Chain Issues Review” with a bullet-point agenda. 
Generate a meeting summary and Action Plan based on the provided “Teams Transcript”. The Action Plan must include:
General Strategies
Immediate Actions
Short-Term Actions (1–3 weeks)
Ongoing Actions

Task 3:
The document generated, Supply Chain Issue Analysis Report, contains:
Executive Summary, Component Overview, Compliance Implications, Root Cause Analysis, and Mitigation Strategies.

task 4:
business presentations in ppt based on existing Excel and Word documentation.
PowerPoint is used to create Supply Chain Issue Compliance Report.pptx.
- The presentation has the following required elements:
1. Title Slide
2. Summary of Key Issues
3. Data Insights from Excel Analysis
4. Proposed Solutions
5. Implementation Plan
- The slides include Copilot generated speaker notes.
